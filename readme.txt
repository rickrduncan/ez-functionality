== Description ==

A plugin designed to host theme independent code snippets that extend and customize the features of WordPress.


== Installation ==

After you've downloaded and extracted the files:

1. Upload the complete 'ez-functionality' folder to the '/wp-content/plugins/' directory OR install via the plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. And yer done!

Visit the <a href="http://rickrduncan.com/pro/wordpress/functionality-plugin">EZ Functionality</a> for more information.
